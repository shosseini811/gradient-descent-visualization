Understanding Gradient Descent

Gradient descent is an optimization algorithm used to minimize a function by iteratively moving in the direction of steepest descent as defined by the negative of the gradient (derivative).

In this visualization:

  • Blue Curve: Represents our cost function f(x) = x⁴ - 4x² + 5
  • Red Dot: Represents our current position on the curve
  • Green Path: Shows how gradient descent moves toward the minimum
  • Purple Lines: Show the tangent lines at each point

For the function f(x) = x⁴ - 4x² + 5:

  • The derivative (gradient) is f'(x) = 4x³ - 8x
  • This function has two local minima at x = -√2 and x = √2
  • There is a local maximum at x = 0
  • The gradient is zero at x = 0, x = -√2, and x = √2
  • Which minimum the algorithm finds depends on the starting position

Parameters:

Learning Rate

Controls how big of a step we take in the direction of the negative gradient. A higher learning rate means bigger steps, which can lead to faster convergence but might overshoot the minimum.

Iterations

The maximum number of steps the algorithm will take. More iterations allow the algorithm to get closer to the minimum, but too many might be unnecessary.

Starting Point

The initial position on the x-axis. Different starting points may lead to different local minima for functions with multiple minima.
