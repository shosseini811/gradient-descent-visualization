How Gradient Descent Works for f(x) = x²

Gradient descent is an optimization algorithm used to minimize a function by iteratively moving in the direction of steepest descent as defined by the negative of the gradient (derivative).

In this visualization:

  • The Blue Curve: Represents our cost function f(x) = x²
  • The Red Dot: Represents our current position on the curve
  • The Green Path: Shows how gradient descent moves toward the minimum
  • The Purple Lines: Show the tangent lines at each point
  • The Red Arrows: Show the negative tangent direction (downhill direction)

For the function f(x) = x⁴ - 4x² + 5:

  • The derivative (gradient) is f'(x) = 4x³ - 8x
  • This function has two local minima at x = -√2 and x = √2
  • There is a local maximum at x = 0
  • The gradient is zero at x = 0, x = -√2, and x = √2
  • Which minimum the algorithm finds depends on the starting position
  • In this visualization, we follow the negative tangent direction (downhill) rather than the perpendicular gradient direction

Learning Rate: Controls how big of a step we take in the direction of the negative gradient. A higher learning rate means bigger steps.

Iterations: The maximum number of steps the algorithm will take.
